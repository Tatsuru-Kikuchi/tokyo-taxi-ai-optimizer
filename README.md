Setup\n\n```bash\n# Clone the repository\ngit clone https://github.com/Tatsuru-Kikuchi/tokyo-taxi-ai-optimizer.git\ncd tokyo-taxi-ai-optimizer\n\n# Backend setup\ncd backend\npython -m venv venv\nsource venv/bin/activate  # On Windows: venv\\Scripts\\activate\npip install -r requirements.txt\n\n# Frontend setup\ncd ../frontend\nnpm install\nnpm install -g @expo/cli\n\n# Database setup\npsql -U postgres -c \"CREATE DATABASE taxi_optimizer;\"\npython manage.py migrate\n\n# Start development servers\n# Terminal 1: Backend API\ncd backend && uvicorn main:app --reload\n\n# Terminal 2: Mobile app\ncd frontend && expo start\n```\n\n### Configuration\n\n```bash\n# Environment variables\ncp .env.example .env\n\n# Required API keys\nODPT_API_KEY=your_odpt_api_key\nWEATHER_API_KEY=your_weather_api_key\nGOOGLE_MAPS_API_KEY=your_google_maps_key\nDATABASE_URL=postgresql://user:pass@localhost/taxi_optimizer\n```\n\n---\n\n## 💼
