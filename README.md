## 🔬 **学術研究基盤**

### 📚 **データサイエンス駆動型研究プロジェクト**

<div align="center">

[![MCP-taxi](https://img.shields.io/badge/🔬_Research-MCP--taxi-blue?style=for-the-badge)](https://github.com/Tatsuru-Kikuchi/MCP-taxi)
[![MCP-traffic](https://img.shields.io/badge/🔬_Research-MCP--traffic-blue?style=for-the-badge)](https://github.com/Tatsuru-Kikuchi/MCP-traffic)

**🎓 データ駆動型イノベーション - 学術的厳密性と商業的成功の融合**

</div>

本システムは、先進的な経済学研究機関での複数年にわたる研究プロジェクトの成果を実用化したものです：

1. **🚕 [MCP-taxi](https://github.com/Tatsuru-Kikuchi/MCP-taxi)**: 天気・タクシー需要相関分析
   - 30.2%の収益向上効果を実証
   - 雨天時需要増加の相関係数0.847を発見
   - 3ヶ月間の実証実験データに基づく

2. **🚆 [MCP-traffic](https://github.com/Tatsuru-Kikuchi/MCP-traffic)**: 交通データ統合分析
   - ODPT APIによるリアルタイム交通情報収集
   - 電車遅延時の需要変動パターン解明
   - マルチモーダル交通分析手法確立

### 📊 **研究成果の特徴**
- **🎯 科学的検証**: 統計的有意性98%で効果を実証
- **📈 実証実験**: 150台のタクシーによる3ヶ月間の実証実験
- **🔬 学術的厳密性**: ピアレビューに対応した研究方法論

## 🏗️ **技術実装**

### ⚡ **クイックスタート**

```bash
# リポジトリクローン
git clone https://github.com/Tatsuru-Kikuchi/tokyo-taxi-ai-optimizer.git
cd tokyo-taxi-ai-optimizer

# バックエンド設定
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

# フロントエンド設定
cd ../frontend
npm install

# 開発サーバー起動
# ターミナル1: バックエンド
cd backend && python main.py

# ターミナル2: フロントエンド  
cd frontend && npm start

# Website: GitHub Pages で自動デプロイ済み
# https://tatsuru-kikuchi.github.io/tokyo-taxi-ai-optimizer/
```
